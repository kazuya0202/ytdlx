# ytdlx - youtube-dl wrapper command

## Features

Supported multi URL(ID) and file that is enumerated URL(ID).

## Usage (Download format)

> `URL` == `URL | ID | FILE`

### 1. Default

```sh
$ ytdlx [URL]
```

### 2. Audio

```sh
$ ytdlx -a [URL]
$ ytdlx --audio [URL]
```

### 3. Video

```sh
$ ytdlx -v [URL]
$ ytdlx --video [URL]
```
